## 버전 관리
2021-05-31 Ver 0.1
-> 게시물 등록,조회,수정,삭제 / 페이징 / 검색 / 댓글

--------------------------------------------------------------


## 개발 환경
- 언어 : JAVA, HTML, CSS, JAVASCRIPT
- 프레임워크 : Spring framework
- 라이브러리 : jQuery, Lombok
- 데이터베이스 : Oracle
- 서버 : Tomcat

## 구현 기능
- 데이터베이스를 활용한 기본적인 게시물 등록,조회,수정,삭제
- 페이징 처리 및 검색 처리
- Ajax 를 이용한 댓글 처리
